# ShasthoMeds-online-medicine-shop-React-Django
ShasthoMeds is an easy-to-use online medicine delivery service for people in Bangladesh. Users can search for medicines, add them to their cart, and order them from home. Some medicines need a doctorâ€™s prescription, which users can upload for safety issues. The admin checks the prescription before the order is confirmed. ShasthoMeds helps people get the right medicine easily and safely, saving time and making health care more accessible from anywhere.
